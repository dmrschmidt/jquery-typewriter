jQuery typewriter documentation

installing it

To start using jQuery typewriter simply download, unzip it and put the following code into your <head> tag.

<script type="text/javascript" src="./js/jquery.min.js"></script>
<script type="text/javascript" src="./js/jquery.class.js"></script>
<script type="text/javascript" src="./js/typewriter.min.js"></script>

running it

After you did that, all you need to do is take any element you want to use the typewriter with and add the class="typewriter" to it.

That's it? Yes... That's it!

mastering it

Now that you installed it and have your first jQuery typewriter up and runnung, you probably want to know how you can customize it to your liking. And guess what, there are quite a few possibilities for that.

general info

By default, the whole content of all 

class="typewriter"
elements will be written in a typewriter way. However, if you don't want all elements to behave the same or even want elements that don't get animated at all, the only thing you need to do, is to provide these elements with 
class="typewrite"
If there is at least one element as a child of your typewriter with this class, then only these elements will be animated.

All configuration is done using custom HTML5 data attributes. Just add these to any element you want to manipulate and there's no need to touch any line of JS. Sweet, innit?

controlling the start

By default all typewriter text will be instantly written out after the page has completed loading. If you don't want this for a any typewriter, simply add the attribute 

data-autostart="false"
to your typewriter element. It will then be excluded from the list of automatically started typewriters and you have to start it manually.
You can use any (clickable) DOM element to start the animation by simply adding 
class="typewriter-starter" data-start-typewriter="TYPEWRITER_ID"
to the element you want to use. Just replace TYPEWRITER_ID with the actual ID of the typewriter you want to control. It will then be started when the user clicks on that element you specified.

letting it cycle

You'll notice, that by default each typewriter does some random "letter-cycling" animations before actually printing a letter and continuing on to the next one.
Firstly, you can control how many letters shall be cycled before the actual letter is printed. Do this by setting a value for 

data-iterations="20"
You can also completely disable this cycling, by adding to the element 
data-cycling="false"

controlling the speed

Is the typewriter writing too fast or too slow for you? No worries. The delay between printing each letter can be customized by setting the attribute a time value in ms (mili seconds) like so 

data-timeout-letter="15"
And if you want to put some emphasize on a certain part, you can also add a custom pause to that element, that is held before printing it. Simply specify the following attribute for that, again, with a time value in ms 
data-prepause="1000"

let it load

Maybe you want to pretend something is currently loading in the background. And of course, there's help for you. Simply specify 

data-prefill="true"
and let the text show a loading animation before printing the actual text. This loading is indicated by dots being written at the place where the text will appear afterwards.
You can also control the time to wait between each dot, by simply setting a value (in ms) for 
data-timeout-wait="100"
