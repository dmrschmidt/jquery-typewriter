what is jQuery typewriter?

The jQuery typewriter allows you to easily add the effect of an old-shool typewriter writing your text to the page.

It is:

  • highly customizable in it's behaviour and output
  • completely unobtrusive
  • very small in size (less than 4 kB)
  • using latest HTML5 technology
  • yet still compatible with old browsers

And it's free! So download jQuery typewriter right now!
